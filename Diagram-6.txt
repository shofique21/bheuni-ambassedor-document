    ┌─────────────────────────────────────────────────────────────────────────────────┐
    │                              STATE TRANSITION LEGEND                             │
    ├─────────────────────────────────────────────────────────────────────────────────┤
    │                                                                                  │
    │  SUBMITTED → VERIFIED    : Passed duplicate + fraud checks                      │
    │  SUBMITTED → REJECTED    : Failed validation (invalid contact, no consent)      │
    │  SUBMITTED → DUPLICATE   : Email/phone already exists in CRM                    │
    │  VERIFIED → CONTACTED    : CRM outreach made                                    │
    │  CONTACTED → APPLICATION : Student started application form                     │
    │  APPLICATION → OFFER     : University issued offer letter                       │
    │  OFFER → ENROLLED        : Student accepted offer and paid fees                 │
    │  ENROLLED → REWARD_PENDING : System queues reward payment                       │
    │  REWARD_PENDING → REWARD_PAID : Payment successfully processed                  │
    │  ENROLLED → WITHDRAWN    : Student withdrew within cooling-off period           │
    │  WITHDRAWN → REWARD_REVERSED : Clawback of enrolment reward                     │
    │  Any non-terminal → EXPIRED : No progress for 90 days                           │
    │                                                                                  │
    └─────────────────────────────────────────────────────────────────────────────────┘
